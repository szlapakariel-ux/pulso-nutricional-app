# Pulso Nutricional Mobile (`@pulso/pulso-nutricional-mobile`)

**Versión reducida del panel profesional** para el celular de la nutricionista.

## Propósito

Permitir a la profesional resolver tareas frecuentes desde el celular: consultar
pacientes, ver fichas resumidas y atender la bandeja de revisión. Es un
subconjunto de Pulso Nutricional PC; **no** replica toda la edición pesada.

## Estado (MC-1)

**Solo placeholder / documentación.** En MC-1 **no** se construye una app mobile
real. El workspace existe con un placeholder TypeScript para mantener la
estructura coherente y verificable, pero sin UI ni funcionalidad.

## Pendiente (microciclos futuros)

- **MC-11**: se define e implementa la versión mobile reducida del profesional,
  decidiendo en ese momento el enfoque tecnológico y reutilizando la API y los
  componentes compartidos.

> No usar datos reales. No conectar Railway. Sin variables de entorno ni secretos.
